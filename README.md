# AI-Generated-Image-Detection
Detection of AI-Generated Images using Gated Expert CNN with Attention Mechanism
Overview

This project detects AI-generated images using a deep learning framework based on ResNet-50 expert networks and an Attention-Based Mixture of Experts (MoE) architecture.

Features
ResNet-50 Transfer Learning
Attention-Based Gating Mechanism
Mixture of Experts (MoE)
AI-Generated Image Classification
Multi-Generator Detection
Dataset

GenImage Dataset

Results
Accuracy: 99.41%
Outperformed baseline CNN and ensemble models
Technologies Used
Python
TensorFlow / Keras
NumPy
Pandas
Scikit-learn
